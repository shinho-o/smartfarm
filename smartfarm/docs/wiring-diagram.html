SmartFarm Sensor Wiring Diagram

Freenove ESP32-S3 WROOM + Sensors

CDS Light Sensor

Analog light intensity (0-100%)
VCC → 3.3V
GND → GND
OUT → GPIO 1 (ADC)

pH Sensor

Water acidity (0-14 pH)
VCC → 3.3V
GND → GND
OUT → GPIO 3 (ADC)

DS18B20 Water Temp

Waterproof temperature (OneWire)
VCC → 3.3V
GND → GND
DATA → GPIO 2 (+ 4.7kΩ pullup)

Data Pipeline

ESP32-S3 센서 수집 + 카메라
→
Mosquitto MQTT Broker (RPi)
→
Telegraf MQTT → InfluxDB (RPi)
→
InfluxDB Time-series DB (RPi)
→
Grafana Dashboard (RPi:3000)
ESP32 Camera :81/stream
→
Browser / Grafana 직접 접속 (MJPEG)

Notes

  • DS18B20의 DATA 핀에 4.7kΩ 풀업 저항 필요 (DATA ↔ 3.3V 사이)
  • SHT31 I2C 주소: 0x44 (기본값)
  • OV2640은 FPC 리본 케이블로 보드 뒷면에 직접 연결
  • 카메라가 GPIO 4~18 대부분을 사용하므로 센서는 GPIO 1, 2, 3, 47, 48 사용
  • 모든 센서 전원: 3.3V (5V 아님 주의)
  • 라즈베리파이 서비스: Mosquitto, Telegraf, InfluxDB, Grafana, Node-RED
